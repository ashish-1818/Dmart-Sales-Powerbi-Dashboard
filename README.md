# 🛒 DMart Sales Analysis Dashboard (Power BI)

## 🔍 Project Overview

This project focuses on analyzing retail sales data using Power BI to uncover trends in sales performance, discounts, and product categories.

## 🎯 Objective

* Analyze sales trends over time
* Evaluate discount effectiveness
* Identify top-performing categories

## 🛠 Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling

## 📊 Dashboard Features

* KPI cards for revenue and sales metrics
* Category-wise performance analysis
* Discount vs Sales impact visualization
* Interactive filters and slicers

## 📈 Key Insights

* Certain discount ranges significantly boost sales
* Seasonal trends impact revenue generation
* Top categories contribute majority of sales
